# EzPrompt
**EzPrompt** is a lightweight, open-source tool designed to help beginners and "Vibe Coders" craft high-quality prompts instantly. Stop worrying about prompt engineering‚Äîjust fill in the blanks, copy, and paste to your favorite LLM (ChatGPT, Claude, Gemini,DeepSeek&ETC).

---

**Features**
- **Beginner Friendly**: No complex syntax. If you can type, you can prompt.
- **One-Click Copy**: Streamlined workflow to get your prompts ready for AI.
- **Pure HTML/JS**: No installation required. Runs directly in your browser.
- **Vibe-Oriented**: Focus on your ideas, let ezprompt handle the structure.

1. **Download** the `EzPrompt.html` file from this repository.
2. **Open** it with any web browser (Chrome, Edge, Safari).

4. copy the result to your AI assistant.
